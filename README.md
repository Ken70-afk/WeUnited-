# WeUnited

WeUnited is a premium professional networking and membership platform built with React. It serves as a modern portal for members, offering features such as user profiling, onboarding, dashboard analytics, success stories, and more.

## Features

- **User Authentication**: Secure Login and Registration flows.
- **Onboarding Experience**: Guided onboarding for new members to set up their profiles.
- **Member Dashboard**: Personalized dashboard with relevant metrics and navigation.
- **Profiles**: View and manage member profiles.
- **Membership Management**: Subscription options and payment success handling.
- **Responsive Design**: Mobile-first architecture ensures a great experience on any device.
- **Premium UI**: Modern aesthetics featuring responsive layouts, custom CSS, and Lucide React icons.

## Tech Stack

- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/weunited.git
   ```

2. Navigate to the project directory:
   ```bash
   cd WeUnited
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

To create a production build:

```bash
npm run build
```

This will generate a `dist` folder containing the optimized and minified assets ready for deployment.
